# Culqui Fullstack Interview

## Usage

### Local development

To execute the function in your local:

1. To install the dependencies
    ```
    $ npm install
    ```

2. To execute the code
    ```
    $ npm run dev
    ```

### Tests

To execute the test

```
$ npm run test
```

### Deployment

To deploy the function into AWS Lambda

```
$ npm run deploy
```
